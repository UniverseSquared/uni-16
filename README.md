# UNI-16

Small WIP fantasy computer inspired by [LIKO-12](https://github.com/RamiLego4Game/LIKO-12).

## Building

If you wish to build UNI-16, you can use the provided Makefile.

```
$ git clone https://github.com/UniverseSquared/uni-16.git
$ cd uni-16/src
$ make
$ cd ../build
```

## Testing

```
$ git clone https://github.com/UniverseSquared/uni-16.git
$ cd uni-16/src
$ love .
```
